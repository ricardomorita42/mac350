# How to build the DB
Inside psql, run:

`\i SUPPORT.sql`

To clean the database:

`\i SUPPORT_CLEAN.sql`


Roles that will be created (so don't have roles with these names):
```
dba
aluno
professor
admin
guest
```

DBs that will be created (So don't have DBs with these names):
```
inter_mod_ace_pes
inter_mod_pes_cur
modulo_acesso
modulo_curriculo
modulo_pessoa
```
